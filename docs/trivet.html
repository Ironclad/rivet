Trivet

Trivet is a library for running Rivet tests, and an integrated test runner in the Rivet application.

For getting started with Trivet Tests, see the Trivet - Getting Started guide.

If you are interested in the Trivet library for running Trivet tests programatically, such as in CI, see the Trivet Library guide.

See the Trivet Tutorial for a step-by-step guide on using Trivet for the first time.
